# HTML-and-CSS-Projects

This repository features nature-inspired web designs using Bootstrap and custom CSS.

## Project 1: Forest Cinema (Bootstrap)
A serene, green-themed movie site.
* **Theme:** "Forest Glass" (Green/White/Transparency).
* **Specs:**
    * **Navbar:** Forest green fixed navigation.
    * **Card Deck:** Shadowed cards with hover lift effects.
    * **Movies:** *Kong, Ghost in the Shell, Beauty and the Beast*.
    * **Badges:** Green "Info" badges for showtimes.

## Project 2: Organic Portfolio
A portfolio focusing on natural aesthetics and glassmorphism.
* **CSS Features:**
    * **Glass Navbar:** Semi-transparent green background with blur (`backdrop-filter`).
    * **Leaf Shapes:** Custom border-radius (`50px 0 50px 0`) on images.
    * **Filters:** `hue-rotate` applied to photography.
    * **Layout:** Flexbox centered layout.
    * **Video:** Nature background loop with text overlay.
